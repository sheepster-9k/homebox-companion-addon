# Homebox Companion Add-on

HA add-on for Homebox Companion (AI-powered inventory assistant).

## Installation

Add this repository URL in **Settings > Add-ons > Repositories**:

```
https://github.com/thesheep120/homebox-companion-addon
```

Then install **Homebox Companion** from the add-on store.

## Configuration

- **Homebox URL** -- URL of your Homebox instance
- **LLM settings** -- Provider, model, and API key for AI-powered features
